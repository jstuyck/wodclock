## About Wodclock

Work Of the Day clock is your perfect buddy when pumping Iron. 

Its mission is to count time while you suffer pulling, pushing, sweating.


## Getting Started

### Installing

To build `wodclock` :

```bash
$ npm install
$ npm install --save @angular/cli
$ ng serve --load
# or
$ npm install
$ npm install --save @angular/cli
$ ng build --prod
```
